# Homebrew Tap

Homebrew formulae for [imamik](https://github.com/imamik) projects.

## Installation

```bash
brew tap imamik/tap
brew install k8zner
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [k8zner](https://github.com/imamik/k8zner) | Production-ready Kubernetes clusters on Hetzner Cloud using Talos Linux |

## Updating

```bash
brew update
brew upgrade k8zner
```
